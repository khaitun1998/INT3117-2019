## **Bước 3: Lập Phương trình của các đường đi**

### Đường 1:
    x = 0, x < n 
    mat[x][j]==no/mat[i][x]==no
    x++
    false

### Đường 2: 
    x = 0, x < n
    mat[x][j]!=no/mat[i][x]!=no
    true


### Đường 3;
    x > n
    sx = (i/3)*3;
    sy = (j/3)*3;
    x = sx, x < sx + 3
    y = sy, y < sy + 3
    mat[x][y] == no
    y++
    x++
    false

### Đường 4 
    x > n
    sx = (i/3)*3;
    sy = (j/3)*3;
    x = sx, x < sx + 3
    y = sy, y < sy + 3
    mat[x][y] /= no
    false

### Đường 5 
    x > n
    sx = (i/3)*3;
    sy = (j/3)*3;
    x = sx, x < sx + 3
    y = sy, y > sy + 3
    true

### Đường 6
    x > n
    sx = (i/3)*3;
    sy = (j/3)*3;
    x = sx, x > sx + 3
    true
